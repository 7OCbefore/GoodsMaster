# 性能优化专项测试

## 优化内容总结

### 1. 列表虚拟化
- **InventoryView**: 已使用 `@tanstack/vue-virtual` 实现虚拟滚动，仅渲染视口内的库存列表项
- **SalesView**: 已使用 `@tanstack/vue-virtual` 实现商品网格虚拟滚动，提升大量商品时的渲染性能

### 2. 构建优化
- **路由懒加载**: 所有主页面路由都已实现懒加载
- **动态导入**: useExport 中的 xlsx 库已实现动态导入，仅在用户点击导出时加载

### 3. PWA 支持
- 已配置 `vite-plugin-pwa` 实现渐进式 Web 应用
- 支持离线访问和添加到主屏幕

## 验证步骤

### 性能测试
1. 在 InventoryView 中加载大量数据（2000+ SKU）
2. 在 SalesView 中加载大量数据（5000+ 商品）
3. 滚动时检查是否保持 60 FPS

### 包体积测试
1. 验证初始加载时 xlsx 库不被加载
2. 点击导出功能时动态加载 xlsx 库

### PWA 功能测试
1. 检查应用可以添加到主屏幕
2. 断网情况下仍可访问应用
